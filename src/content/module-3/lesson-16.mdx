## 1. üìò Deep Theory: Compressing Language

In spoken German (Verbal Style), we use subordinate clauses with connectors (*weil, wenn, obwohl*).
In formal/academic German (Nominal Style), we turn verbs into nouns and use prepositions.

### The Transformation Mechanism

**Goal:** Say the same thing with fewer words, but more density.

* **Verbal (B1):** "Wir konnten nicht arbeiten, **weil** der Strom **ausfiel**." (Connector + Verb).
* **Nominal (B2/C1):** "**Wegen** des **Stromausfalls** konnten wir nicht arbeiten." (Preposition + Noun).

### The "Magic Mapping" Table

You must memorize which Connector becomes which Preposition.

| Logic | Verbal Connector (Nebensatz) | Nominal Preposition (+ Genitiv/Dat) |
| --- | --- | --- |
| **Causal** (Reason) | weil / da | **wegen** (+ Gen) / **aufgrund** (+ Gen) |
| **Concessive** (Counter-argument) | obwohl | **trotz** (+ Gen) |
| **Temporal** (Time) | als / wenn / w√§hrend | **bei** (+ Dat) / **w√§hrend** (+ Gen) |
| **Modal** (Method) | indem / dadurch, dass | **durch** (+ Akk) |
| **Final** (Goal) | um ... zu / damit | **zu** (+ Dat) / **zwecks** (+ Gen) |
| **Conditional** (Condition) | wenn / falls | **bei** (+ Dat) |
